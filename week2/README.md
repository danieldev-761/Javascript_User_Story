# Week 2: Data Management with Objects, Sets, and Maps

This project focuses on handling complex data structures in JavaScript, including Objects, Sets, and Maps, and how to iterate through them efficiently.

## Objectives
- Create and manipulate JavaScript Objects.
- Use `Set` to handle unique data and prevent duplicates.
- Use `Map` to store associative information (Key-Value pairs).
- Master different iteration methods: `for...in`, `for...of`, and `forEach()`.
- Use Object methods like `Object.keys()`, `Object.values()`, and `Object.entries()`.

## Key Tasks
1. **Product Objects**: A collection of items with ID, name, and price.
2. **Unique Sets**: demonstrating how Sets automatically remove duplicates from a list of numbers.
3. **Product Maps**: Mapping product categories to specific product names.
4. **Data Validation**: A simple function to ensure all products have valid IDs, names, and prices before displaying them.

## How to use
1. Open `index.html` in your web browser.
2. Open the **Console (F12)** to view the categorized outputs for each task.
3. Observe how the validation logic filters out incomplete or incorrect data.

## Project Structure
- `index.html`: Entry point to execute the script.
- `gestion_datos.js`: The main logic containing all tasks and data structures.
