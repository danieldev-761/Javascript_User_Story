# Week 1: Interactive Greeting System

A simple interactive program that gathers user information and provides a personalized message based on age.

## Objectives
- Practice basic JavaScript interaction using `alert()`, `prompt()`, and `console.log()`.
- Implement simple conditional logic (`if/else`).
- Handle data types (converting strings to numbers).
- Perform basic input validation.

## Features
1. **Personalized Welcome**: Greets the user and asks for their name and age.
2. **Age Verification**: Checks if the user is 18 or older (legal age).
3. **Dynamic Feedback**: Provides different encouraging messages based on the user's age.
4. **Input Handling**: Detects empty inputs or invalid numbers to prevent errors.

##  How to use
1. Open `index.html` in your web browser.
2. Follow the pop-up prompts (Alerts and Prompts).
3. Check the **Console (F12)** to see the verification logs.

##  Project Structure
- `index.html`: Entry point.
- `sistema_interactivo.js`: Main JavaScript logic.
